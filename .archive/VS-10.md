# Implementation Plan

This plan is the code we intend to write, expressed in Markdown first.

The vertical slice is complete when this command works:

```bash
cargo run -p indexer -- explain interaction.checkout.create-order
```

The command must:

- scan `.docs/**/*.md` Markdown frontmatter for soul document IDs
- scan Rust and C# files for soul attributes
- link documents and annotations by exact `id`
- print matching docs and code references
- survive malformed content by emitting diagnostics instead of crashing

This slice uses native Rust/C# attributes as the authoring surface for code annotations.
The older `// @soul {...}` comment format remains documented elsewhere in the repo but is not implemented by this vertical slice.

Rust attribute crate responsibilities:

- expose `#[proc_macro_attribute] pub fn soul(...)`
- accept an attribute payload with arbitrary named fields and require `id`
- return the input item unchanged so the attribute is dev-time metadata only in this slice
- use typed errors internally instead of ad hoc strings
- capture `ErrorLocation` via helper constructors before converting typed failures into compile errors, but keep the emitted compiler span attached to the attribute payload so the user-facing diagnostic still points at `#[soul(...)]`
- reject duplicate keys, non-string literal values, and empty required values with normal compile errors

C# package responsibilities:

- define `public sealed class SoulAttribute : Attribute`
- require `id` through the constructor and expose it as immutable after construction
- use the constructor positionally in source, so the real attribute form is `[Soul("...", Role = "...")]` rather than a fake `Id =` named argument
- treat C# extensibility as an explicit boundary: `Role` and `MetadataJson` may be first-class properties, and arbitrary additional metadata for this slice is carried through one validated `MetadataJson` object payload
- reject duplicate keys inside `MetadataJson` before the indexer normalizes that payload so malformed metadata cannot silently collapse to a last-write-wins shape
- support usage on methods, classes, records, and other declaration-level surfaces used by the repo
- carry no business logic beyond metadata presence; any validation happens eagerly when the attribute is instantiated or reflected, not in the indexer
- centralize validation in named guard methods instead of ad hoc inline checks
- reject null, empty, or whitespace-only required values eagerly
- normalize scalar optional string properties consistently so whitespace-only values do not survive as meaningful state
- treat `MetadataJson` as validated JSON payload rather than a scalar optional string, so blank content is invalid instead of silently disappearing
- reject malformed metadata JSON eagerly
- reject metadata JSON whose root value is not an object so arbitrary metadata has one predictable shape
- the indexer, not the C# package, unpacks validated metadata JSON properties into the same normalized metadata map used by Rust annotations
- the C# package is a source-authored contract, but its validation only runs when the attribute is instantiated or reflected; it does not alter source parsing or provide compile-time enforcement
- document that this contract is intentionally stricter than the Rust proc-macro surface because C# attributes have different language constraints

Validation proof points for these authoring dependencies:

- Rust: compile-pass and compile-fail style tests in `crates/soul-attributes/tests` should prove a valid attribute expands to the original item unchanged by actually using the annotated item in the pass case with `use soul_attributes::soul;`, and that non-identifier keys, duplicate keys, non-string values, empty required values, and missing `id` are rejected, with committed `.stderr` baselines for failures. Every trybuild UI fixture should import the proc macro crate explicitly so the pass and fail cases are both self-contained.
- C#: unit tests in `packages/Soul.Attributes.Tests` should prove constructor validation, `Role` normalization, whitespace-only `MetadataJson` rejection, invalid JSON rejection, duplicate metadata-key rejection, reserved-key rejection, and non-object metadata rejection, and the test project should reference `packages/Soul.Attributes/Soul.Attributes.csproj` directly. The test source must import both `Soul.Attributes` and `Xunit` or use equivalent aliases so `[Soul(...)]` and `[Fact]` resolve, and the project also needs an explicit test SDK and framework package so `dotnet test` is runnable.

The indexer still parses source text directly.
It must not depend on macro expansion or reflected assemblies.
Examples may show `role`, but the annotation schema for this system remains open-ended overall.
For C#, that openness is expressed through validated JSON object metadata rather than arbitrary attribute argument shapes.
This slice only accepts standard quoted string literals in the source-text scanner; raw Rust strings and C# verbatim strings are intentionally out of scope.
Rust string literals are decoded with Rust syntax rules, and C# regular string literals are decoded with C# escape rules instead of JSON rules so valid source escapes like `\x`, `\u`, `\U`, `\0`, and `\n` survive scanner normalization.
Optional trailing commas after the final named argument are accepted so the scanner matches the proc-macro and C# attribute parsers.
Prefix lookalikes such as `#[soulful(...)]` and `[Soulful(...)]` are ignored rather than diagnosed so the scanner only reports exact attribute forms.
The workspace root `Cargo.toml` is updated so `cargo run -p indexer` and `cargo test -p indexer` can target the workspace members.

## `crates/soul-attributes/src/lib.rs`

This proc-macro should accept arbitrary soul key/value pairs, use typed internal errors, and then convert those failures into compile errors at the attribute site.

```rust
use proc_macro::TokenStream;
use quote::quote;
use syn::parse::Parser;
use syn::punctuated::Punctuated;
use syn::{parse_macro_input, Error, Expr, ExprLit, Item, Lit, MetaNameValue, Token};
use std::collections::BTreeMap;

mod error;

use crate::error::{Result, SoulAttributeError};

fn compile_error(spanned: impl quote::ToTokens, error: SoulAttributeError) -> TokenStream {
    Error::new_spanned(spanned, error.to_string())
        .to_compile_error()
        .into()
}

fn parse_fields(args: &Punctuated<MetaNameValue, Token![,]>) -> Result<BTreeMap<String, String>> {
    let mut fields = BTreeMap::<String, String>::new();

    for arg in args {
        let Some(ident) = arg.path.get_ident() else {
            return Err(SoulAttributeError::non_identifier_key());
        };

        let key = ident.to_string();

        let value = match &arg.value {
            Expr::Lit(ExprLit { lit: Lit::Str(value), .. }) => value.value(),
            _ => return Err(SoulAttributeError::non_string_value()),
        };

        if key == "id" && value.trim().is_empty() {
            return Err(SoulAttributeError::empty_value(&key));
        }

        if fields.insert(key.clone(), value).is_some() {
            return Err(SoulAttributeError::duplicate_field(&key));
        }
    }

    if !fields.contains_key("id") {
        return Err(SoulAttributeError::missing_id());
    }

    Ok(fields)
}

#[proc_macro_attribute]
pub fn soul(args: TokenStream, item: TokenStream) -> TokenStream {
    let parser = Punctuated::<MetaNameValue, Token![,]>::parse_terminated;
    let args = parse_macro_input!(args with parser);
    let parsed_item = parse_macro_input!(item as Item);

    if let Err(error) = parse_fields(&args) {
        return compile_error(&args, error);
    }

    quote!(#parsed_item).into()
}
```

## `packages/Soul.Attributes/SoulAttribute.cs`

This attribute type should make the C# authoring surface explicit, require a non-empty `id`, and fail fast through named guard methods instead of raw inline shortcuts.
The plan should be honest about C# attribute constraints: we are aiming for equivalent rigor to Rust, not fake syntax symmetry.
The production boundary for arbitrary extra metadata in this slice is one validated JSON object payload with unique top-level property names.

```csharp
using System;
using System.Collections.Generic;
using System.Text.Json;

namespace Soul.Attributes;

[AttributeUsage(
    AttributeTargets.Class |
    AttributeTargets.Struct |
    AttributeTargets.Interface |
    AttributeTargets.Enum |
    AttributeTargets.Method |
    AttributeTargets.Property,
    AllowMultiple = true,
    Inherited = false)]
public sealed class SoulAttribute : Attribute
{
    private string? _role;
    private string? _metadataJson;

    public SoulAttribute(string id)
    {
        Id = RequireNonEmpty(id, nameof(id));
    }

    public string Id { get; }

    public string? Role
    {
        get => _role;
        set => _role = NormalizeOptional(value);
    }

    // Carries arbitrary extra metadata for this slice until a richer C# authoring DSL exists.
    // The payload must be a JSON object with unique property names so the extensibility boundary has one predictable shape.
    // Duplicate properties are rejected before the normalized metadata map is accepted.
    // Example: "{\"surface\":\"checkout\",\"capability\":\"orders\"}"
    public string? MetadataJson
    {
        get => _metadataJson;
        set => _metadataJson = NormalizeAndValidateMetadata(value, nameof(MetadataJson));
    }

    private static string RequireNonEmpty(string value, string paramName)
    {
        if (string.IsNullOrWhiteSpace(value))
        {
            throw new ArgumentException(
                "Value must not be null, empty, or whitespace.",
                paramName);
        }

        return value.Trim();
    }

    private static string? NormalizeOptional(string? value)
        => string.IsNullOrWhiteSpace(value) ? null : value.Trim();

    private static string? NormalizeAndValidateMetadata(string? value, string paramName)
    {
        if (value is null)
        {
            return null;
        }

        var normalized = value.Trim();
        if (normalized.Length == 0)
        {
            throw new ArgumentException(
                "Soul attribute metadata must be valid JSON.",
                paramName);
        }

        try
        {
            using var document = JsonDocument.Parse(normalized);
            if (document.RootElement.ValueKind != JsonValueKind.Object)
            {
                throw new ArgumentException(
                    "Soul attribute metadata must be a JSON object.",
                    paramName);
            }

            var seen = new HashSet<string>(StringComparer.Ordinal);
            foreach (var property in document.RootElement.EnumerateObject())
            {
                if (property.Name is "id" or "role")
                {
                    throw new ArgumentException(
                        "Soul attribute metadata must not use reserved property names.",
                        paramName);
                }

                if (!seen.Add(property.Name))
                {
                    throw new ArgumentException(
                        "Soul attribute metadata must not contain duplicate property names.",
                        paramName);
                }
            }
        }
        catch (JsonException ex)
        {
            throw new ArgumentException(
                "Soul attribute metadata must be valid JSON.",
                paramName,
                ex);
        }

        return normalized;
    }
}
```

## `crates/indexer/src/error.rs`

Fatal failures use `IndexerError` for invalid roots, traversal failures, and CLI argument errors.

Recoverable content problems do not use `IndexerError`.
They become `Diagnostic` values attached to the scan result.
Per-file read failures also become diagnostics so a single unreadable file does not abort the repository-wide scan.
When WalkDir reports a nested entry failure, prefer the failing entry path when available instead of collapsing everything to the scan root.
Capture `ErrorLocation` at the call site by using the helper constructors above instead of constructing enum variants directly, but keep it out of the formatted error message so CLI output stays focused on the failing repository path or argument.

There is no `anyhow` in this crate.

```rust
use error_location::ErrorLocation;
use std::path::PathBuf;
use std::panic::Location;

#[derive(Debug, thiserror::Error)]
pub enum IndexerError {
    #[error("root does not exist or is not a directory: {path}")]
    InvalidRoot {
        path: PathBuf,
        location: ErrorLocation,
    },

    #[error("failed to walk path {path}: {source}")]
    WalkEntry {
        path: PathBuf,
        location: ErrorLocation,
        #[source]
        source: std::io::Error,
    },

    #[error("{message}")]
    Cli {
        message: String,
        location: ErrorLocation,
    },
}

pub type Result<T> = std::result::Result<T, IndexerError>;

impl IndexerError {
    #[track_caller]
    pub fn invalid_root(path: PathBuf) -> Self {
        Self::InvalidRoot {
            path,
            location: ErrorLocation::from(Location::caller()),
        }
    }

    #[track_caller]
    pub fn walk_entry(path: PathBuf, source: std::io::Error) -> Self {
        Self::WalkEntry {
            path,
            location: ErrorLocation::from(Location::caller()),
            source,
        }
    }

    #[track_caller]
    pub fn cli(message: impl Into<String>) -> Self {
        Self::Cli {
            message: message.into(),
            location: ErrorLocation::from(Location::caller()),
        }
    }
}
```

## `crates/indexer/src/markdown.rs`

Implementation notes:

- Markdown body is ignored in this slice.
- A Markdown file without frontmatter is not an error.
- A Markdown file with malformed or incomplete frontmatter is a diagnostic.
- Duplicate frontmatter keys are malformed input and must not be allowed to collapse to a last-write-wins mapping before `Frontmatter` is built.
- Duplicate document IDs across `.docs` files are malformed input and must emit a diagnostic; resolve them after collecting and sorting documents by relative path so the lexicographically first path wins deterministically.
- A closing frontmatter fence at EOF is valid.
- `Err(IndexerError)` is reserved for fatal parser failures; malformed user content stays in `ParseReport.diagnostics`.
Implement `parse_unique_frontmatter(frontmatter_text)` in `markdown.rs` as a duplicate-aware YAML helper rather than calling `serde_yaml::from_str::<Frontmatter>` directly.
The helper should deserialize the top-level mapping key by key, reject repeated keys before any collapse to `Frontmatter`, and return ordinary YAML syntax failures as the same recoverable frontmatter diagnostic path used by `parse_markdown`.

```rust
use std::path::Path;

use serde::Deserialize;

use crate::error::Result;
use crate::model::{Diagnostic, DiagnosticSeverity, Document, ParseReport};

#[derive(Debug, Deserialize)]
struct Frontmatter {
    id: Option<String>,
    kind: Option<String>,
    title: Option<String>,
}

pub fn parse_markdown(path: &Path, input: &str) -> Result<ParseReport<Option<Document>>> {
    let normalized = input.replace("\r\n", "\n");
    let report = match extract_frontmatter(&normalized) {
        FrontmatterBlock::Absent => ParseReport {
            value: None,
            diagnostics: Vec::new(),
        },
        FrontmatterBlock::Unterminated => ParseReport {
            value: None,
            diagnostics: vec![Diagnostic {
                severity: DiagnosticSeverity::Error,
                path: path.to_path_buf(),
                line: None,
                message: "frontmatter block is missing a closing `---` delimiter".to_string(),
            }],
        },
        FrontmatterBlock::Present(frontmatter_text) => {
            // Parse through a duplicate-aware helper so repeated frontmatter keys become
            // diagnostics instead of silently collapsing to the last value.
            let parsed = match parse_unique_frontmatter(frontmatter_text) {
                Ok(parsed) => parsed,
                Err(error) => {
                    return Ok(ParseReport {
                        value: None,
                        diagnostics: vec![Diagnostic {
                            severity: DiagnosticSeverity::Error,
                            path: path.to_path_buf(),
                            line: None,
                            message: format!("invalid frontmatter: {error}"),
                        }],
                    });
                }
            };

            match (parsed.id, parsed.kind) {
                (Some(id), Some(kind)) if !id.trim().is_empty() && !kind.trim().is_empty() => {
                    ParseReport {
                        value: Some(Document {
                            id: id.trim().to_string(),
                            kind: kind.trim().to_string(),
                            title: parsed
                                .title
                                .and_then(|title| {
                                    let title = title.trim();
                                    (!title.is_empty()).then(|| title.to_string())
                                }),
                            path: path.to_path_buf(),
                        }),
                        diagnostics: Vec::new(),
                    }
                }
                _ => ParseReport {
                    value: None,
                    diagnostics: vec![Diagnostic {
                        severity: DiagnosticSeverity::Error,
                        path: path.to_path_buf(),
                        line: None,
                        message: "frontmatter must include non-empty required fields `id` and `kind`"
                            .to_string(),
                    }],
                },
            }
        }
    };

    Ok(report)
}

enum FrontmatterBlock<'a> {
    Absent,
    Unterminated,
    Present(&'a str),
}

fn extract_frontmatter(input: &str) -> FrontmatterBlock<'_> {
    let Some(rest) = input.strip_prefix("---\n") else {
        return FrontmatterBlock::Absent;
    };

    if let Some(end) = rest.find("\n---\n") {
        return FrontmatterBlock::Present(&rest[..end]);
    }

    if let Some(frontmatter) = rest.strip_suffix("\n---") {
        return FrontmatterBlock::Present(frontmatter);
    }

    FrontmatterBlock::Unterminated
}

fn parse_unique_frontmatter(frontmatter_text: &str) -> std::result::Result<Frontmatter, String> {
    use serde::de::{Deserializer as _, MapAccess, Visitor};
    use std::fmt;

    struct UniqueMapVisitor;

    impl<'de> Visitor<'de> for UniqueMapVisitor {
        type Value = std::collections::BTreeMap<String, serde_yaml::Value>;

        fn expecting(&self, formatter: &mut fmt::Formatter<'_>) -> fmt::Result {
            formatter.write_str("a YAML mapping with unique keys")
        }

        fn visit_map<A>(self, mut map: A) -> std::result::Result<Self::Value, A::Error>
        where
            A: MapAccess<'de>,
        {
            let mut fields = std::collections::BTreeMap::new();
            while let Some((key, value)) = map.next_entry::<String, serde_yaml::Value>()? {
                if fields.insert(key.clone(), value).is_some() {
                    return Err(serde::de::Error::custom(format!(
                        "duplicate frontmatter key `{key}`"
                    )));
                }
            }
            Ok(fields)
        }
    }

    let deserializer = serde_yaml::Deserializer::from_str(frontmatter_text);
    let raw_map = deserializer
        .deserialize_map(UniqueMapVisitor)
        .map_err(|e| e.to_string())?;

    let id = raw_map
        .get("id")
        .and_then(|v| v.as_str())
        .map(str::to_string);
    let kind = raw_map
        .get("kind")
        .and_then(|v| v.as_str())
        .map(str::to_string);
    let title = raw_map
        .get("title")
        .and_then(|v| v.as_str())
        .map(str::to_string);

    Ok(Frontmatter { id, kind, title })
}
```

```rust
use std::path::Path;

use serde_json::{Map, Value};

use crate::error::Result;
use crate::model::{
    AnnotationSyntax, CodeAnnotation, Diagnostic, DiagnosticSeverity, ParseReport,
};

#[derive(Debug, Clone, PartialEq, Eq)]
struct NormalizedAnnotation {
    id: String,
    role: Option<String>,
    metadata: Map<String, Value>,
    raw: String,
}

trait AnnotationParser {
    fn syntax(&self) -> AnnotationSyntax;
    fn parse_line(&self, line: &str) -> Option<std::result::Result<NormalizedAnnotation, String>>;
}

struct RustAttributeParser;
struct CSharpAttributeParser;

static RUST_ATTRIBUTE_PARSER: RustAttributeParser = RustAttributeParser;
static CSHARP_ATTRIBUTE_PARSER: CSharpAttributeParser = CSharpAttributeParser;

pub fn parse_annotations(path: &Path, input: &str) -> Result<ParseReport<Vec<CodeAnnotation>>> {
    let Some(parser) = parser_for_path(path) else {
        return Ok(ParseReport {
            value: Vec::new(),
            diagnostics: Vec::new(),
        });
    };

    let mut annotations = Vec::new();
    let mut diagnostics = Vec::new();

    for (index, line) in input.lines().enumerate() {
        let Some(parsed) = parser.parse_line(line) else {
            continue;
        };

        match parsed {
            Ok(annotation) => annotations.push(CodeAnnotation {
                id: annotation.id,
                role: annotation.role,
                metadata: annotation.metadata,
                path: path.to_path_buf(),
                line: index + 1,
                syntax: parser.syntax(),
                raw: annotation.raw,
            }),
            Err(message) => diagnostics.push(Diagnostic {
                severity: DiagnosticSeverity::Error,
                path: path.to_path_buf(),
                line: Some(index + 1),
                message,
            }),
        }
    }

    Ok(ParseReport {
        value: annotations,
        diagnostics,
    })
}

fn parser_for_path(path: &Path) -> Option<&'static dyn AnnotationParser> {
    match path.extension().and_then(|ext| ext.to_str()) {
        Some("rs") => Some(&RUST_ATTRIBUTE_PARSER),
        Some("cs") => Some(&CSHARP_ATTRIBUTE_PARSER),
        _ => None,
    }
}

impl AnnotationParser for RustAttributeParser {
    fn syntax(&self) -> AnnotationSyntax {
        AnnotationSyntax::RustAttribute
    }

    fn parse_line(&self, line: &str) -> Option<std::result::Result<NormalizedAnnotation, String>> {
        let trimmed = line.trim();
        if !trimmed.starts_with("#[soul(") {
            return None;
        }

        if !trimmed.ends_with(")]") {
            return Some(Err("malformed soul attribute".to_string()));
        }

        let payload = &trimmed["#[soul(".len()..trimmed.len() - 2];
        Some(parse_rust_fields(payload).and_then(|fields| {
            normalized_annotation_from_fields(fields, trimmed)
        }))
    }
}

impl AnnotationParser for CSharpAttributeParser {
    fn syntax(&self) -> AnnotationSyntax {
        AnnotationSyntax::CSharpAttribute
    }

    fn parse_line(&self, line: &str) -> Option<std::result::Result<NormalizedAnnotation, String>> {
        let trimmed = line.trim();
        if !trimmed.starts_with("[Soul(") {
            return None;
        }

        if !trimmed.ends_with(")]") {
            return Some(Err("malformed soul attribute".to_string()));
        }

        let payload = &trimmed["[Soul(".len()..trimmed.len() - 2];
        Some(parse_csharp_fields(payload).and_then(|fields| {
            normalized_annotation_from_fields(fields, trimmed)
        }))
    }
}

fn parse_rust_fields(payload: &str) -> std::result::Result<Map<String, Value>, String> {
    let mut fields = Map::new();

    for segment in split_assignments(payload)? {
        let (key, value) = parse_rust_assignment(&segment)?;
        insert_metadata_field(&mut fields, key, Value::String(value))?;
    }

    Ok(fields)
}

fn parse_csharp_fields(payload: &str) -> std::result::Result<Map<String, Value>, String> {
    let mut segments = split_assignments(payload)?.into_iter();
    let Some(first) = segments.next() else {
        return Err("missing required field `id`".to_string());
    };
    if !first.trim_start().starts_with('"') {
        return Err("missing required field `id`".to_string());
    }

    let mut fields = Map::new();
    let mut metadata_json_seen = false;
    insert_metadata_field(
        &mut fields,
        "id".to_string(),
        Value::String(parse_csharp_string_literal(&first)?),
    )?;

    for segment in segments {
        let (key, value) = parse_csharp_assignment(&segment)?;
        match key.as_str() {
            "Role" => insert_metadata_field(&mut fields, "role".to_string(), Value::String(value))?,
            "MetadataJson" => {
                if metadata_json_seen {
                    return Err("malformed soul attribute".to_string());
                }
                metadata_json_seen = true;
                merge_metadata_json(&mut fields, &value)?;
            }
            _ => return Err("malformed soul attribute".to_string()),
        }
    }

    Ok(fields)
}

fn split_assignments(payload: &str) -> std::result::Result<Vec<String>, String> {
    let mut parts = Vec::new();
    let mut current = String::new();
    let mut in_string = false;
    let mut escape = false;

    for ch in payload.chars() {
        if escape {
            current.push(ch);
            escape = false;
            continue;
        }

        match ch {
            '\\' if in_string => {
                current.push(ch);
                escape = true;
            }
            '"' => {
                current.push(ch);
                in_string = !in_string;
            }
            ',' if !in_string => {
                let segment = current.trim();
                if segment.is_empty() {
                    return Err("malformed soul attribute".to_string());
                }
                parts.push(segment.to_string());
                current.clear();
            }
            _ => current.push(ch),
        }
    }

    if in_string {
        return Err("malformed soul attribute".to_string());
    }

    let tail = current.trim();
    if tail.is_empty() {
        if payload.trim().is_empty() {
            return Ok(Vec::new());
        }
        if payload.trim_end().ends_with(',') {
            return Ok(parts);
        }
        return Err("malformed soul attribute".to_string());
    }

    parts.push(tail.to_string());
    Ok(parts)
}

fn parse_rust_assignment(segment: &str) -> std::result::Result<(String, String), String> {
    let (raw_key, raw_value) = segment
        .split_once('=')
        .ok_or_else(|| "malformed soul attribute".to_string())?;

    let key = raw_key.trim();
    let value = raw_value.trim();

    if key.is_empty()
        || syn::parse_str::<syn::Ident>(key).is_err()
        || !value.starts_with('"')
        || !value.ends_with('"')
        || value.len() < 2
    {
        return Err("malformed soul attribute".to_string());
    }

    let inner = decode_rust_string_literal(value)
        .map_err(|_| "malformed soul attribute".to_string())?;

    Ok((key.to_string(), inner))
}

fn parse_csharp_assignment(segment: &str) -> std::result::Result<(String, String), String> {
    let (raw_key, raw_value) = segment
        .split_once('=')
        .ok_or_else(|| "malformed soul attribute".to_string())?;

    let key = raw_key.trim();
    let value = raw_value.trim();

    if key.is_empty() || !value.starts_with('"') || !value.ends_with('"') || value.len() < 2 {
        return Err("malformed soul attribute".to_string());
    }

    match key {
        "Role" | "MetadataJson" => {
            let inner = parse_csharp_string_literal(value)?;
            Ok((key.to_string(), inner))
        }
        _ => Err("malformed soul attribute".to_string()),
    }
}

fn parse_csharp_string_literal(raw_value: &str) -> std::result::Result<String, String> {
    let raw_value = raw_value.trim();

    if !raw_value.starts_with('"') || !raw_value.ends_with('"') {
        return Err("malformed soul attribute".to_string());
    }

    decode_csharp_string_literal(raw_value).map_err(|_| "malformed soul attribute".to_string())
}

fn decode_csharp_string_literal(raw_value: &str) -> std::result::Result<String, String> {
    // Decode the regular C# escape set directly so the scanner matches source syntax
    // instead of narrowing to JSON-compatible escapes.
    let inner = &raw_value[1..raw_value.len() - 1];
    let mut decoded = String::new();
    let mut chars = inner.chars().peekable();

    while let Some(ch) = chars.next() {
        if ch != '\\' {
            decoded.push(ch);
            continue;
        }

        let Some(escape) = chars.next() else {
            return Err("malformed soul attribute".to_string());
        };

        match escape {
            '\\' => decoded.push('\\'),
            '"' => decoded.push('"'),
            '\'' => decoded.push('\''),
            '0' => decoded.push('\0'),
            'a' => decoded.push('\u{0007}'),
            'b' => decoded.push('\u{0008}'),
            'e' => decoded.push('\u{001B}'),
            'f' => decoded.push('\u{000C}'),
            'n' => decoded.push('\n'),
            'r' => decoded.push('\r'),
            't' => decoded.push('\t'),
            'v' => decoded.push('\u{000B}'),
            'x' => decoded.push(parse_csharp_hex_escape(&mut chars, 1, 4)?),
            'u' => decoded.push(parse_csharp_hex_escape(&mut chars, 4, 4)?),
            'U' => decoded.push(parse_csharp_hex_escape(&mut chars, 8, 8)?),
            _ => return Err("malformed soul attribute".to_string()),
        }
    }

    Ok(decoded)
}

fn parse_csharp_hex_escape(
    chars: &mut std::iter::Peekable<std::str::Chars<'_>>,
    min_digits: usize,
    max_digits: usize,
) -> std::result::Result<char, String> {
    let mut digits = String::new();

    while digits.len() < max_digits {
        let Some(next) = chars.peek().copied() else {
            break;
        };

        if !next.is_ascii_hexdigit() {
            break;
        }

        digits.push(next);
        chars.next();
    }

    if digits.len() < min_digits {
        return Err("malformed soul attribute".to_string());
    }

    let value = u32::from_str_radix(&digits, 16).map_err(|_| "malformed soul attribute".to_string())?;
    char::from_u32(value).ok_or_else(|| "malformed soul attribute".to_string())
}

fn decode_rust_string_literal(raw_value: &str) -> std::result::Result<String, String> {
    syn::parse_str::<syn::LitStr>(raw_value)
        .map(|literal| literal.value())
        .map_err(|_| "malformed soul attribute".to_string())
}

fn insert_metadata_field(
    fields: &mut Map<String, Value>,
    key: String,
    value: Value,
) -> std::result::Result<(), String> {
    if fields.insert(key, value).is_some() {
        return Err("malformed soul attribute".to_string());
    }

    Ok(())
}

fn merge_metadata_json(fields: &mut Map<String, Value>, json: &str) -> std::result::Result<(), String> {
    let metadata = parse_unique_json_object(json)?;

    for (key, value) in metadata {
        if key == "id" || key == "role" || fields.contains_key(&key) {
            return Err("malformed soul attribute".to_string());
        }

        fields.insert(key, value);
    }

    Ok(())
}

fn normalized_annotation_from_fields(
    mut fields: Map<String, Value>,
    raw: &str,
) -> std::result::Result<NormalizedAnnotation, String> {
    let id = take_trimmed_string(&mut fields, "id")
        .ok_or_else(|| "missing required field `id`".to_string())?;

    let role = take_trimmed_string(&mut fields, "role");

    Ok(NormalizedAnnotation {
        id,
        role,
        metadata: fields,
        raw: raw.to_string(),
    })
}

fn parse_unique_json_object(json: &str) -> std::result::Result<Map<String, Value>, String> {
    use serde::de::{Deserializer, MapAccess, Visitor};
    use std::fmt;

    struct UniqueObjectVisitor;

    impl<'de> Visitor<'de> for UniqueObjectVisitor {
        type Value = Map<String, Value>;

        fn expecting(&self, formatter: &mut fmt::Formatter<'_>) -> fmt::Result {
            formatter.write_str("a JSON object with unique property names")
        }

        fn visit_map<A>(self, mut map: A) -> std::result::Result<Self::Value, A::Error>
        where
            A: MapAccess<'de>,
        {
            let mut fields = Map::new();
            while let Some((key, value)) = map.next_entry::<String, Value>()? {
                if fields.insert(key, value).is_some() {
                    return Err(serde::de::Error::custom("duplicate property name"));
                }
            }
            Ok(fields)
        }
    }

    let mut deserializer = serde_json::Deserializer::from_str(json);
    let parsed = deserializer
        .deserialize_map(UniqueObjectVisitor)
        .map_err(|_| "malformed soul attribute".to_string())?;
    deserializer
        .end()
        .map_err(|_| "malformed soul attribute".to_string())?;
    Ok(parsed)
}

fn take_trimmed_string(fields: &mut Map<String, Value>, key: &str) -> Option<String> {
    match fields.remove(key) {
        Some(Value::String(value)) => {
            let trimmed = value.trim();
            (!trimmed.is_empty()).then(|| trimmed.to_string())
        }
        Some(_) => None,
        None => None,
    }
}
```

```rust
use std::fs;
use std::io::ErrorKind;
use std::path::{Component, Path, PathBuf};

use walkdir::{DirEntry, WalkDir};

use crate::annotation::parse_annotations;
use crate::error::{IndexerError, Result};
use crate::markdown::parse_markdown;
use crate::model::{Diagnostic, DiagnosticSeverity, SemanticGraph};

pub fn scan_repository(root: &Path) -> Result<SemanticGraph> {
    if !root.exists() || !root.is_dir() {
        return Err(IndexerError::invalid_root(root.to_path_buf()));
    }

    let mut graph = SemanticGraph::default();

    for entry in WalkDir::new(root)
        .into_iter()
        .filter_entry(|entry| !is_excluded_dir(root, entry))
    {
        let entry = entry.map_err(|error| {
            IndexerError::walk_entry(
                error
                    .path()
                    .map(|path| path.to_path_buf())
                    .unwrap_or_else(|| root.to_path_buf()),
                error
                    .into_io_error()
                    .unwrap_or_else(|| std::io::Error::other("walkdir error")),
            )
        })?;

        if !entry.file_type().is_file() {
            continue;
        }

        let path = entry.path();
        let Some(candidate) = classify_path(root, path) else {
            continue;
        };

        let contents = match fs::read_to_string(path) {
            Ok(contents) => contents,
            Err(source) => {
                graph
                    .diagnostics
                    .push(read_failure_diagnostic(&candidate.display_path, source.kind(), source));
                continue;
            }
        };

        match candidate.kind {
            CandidateKind::Document => {
                let report = parse_markdown(&candidate.display_path, &contents)?;
                if let Some(document) = report.value {
                    graph.documents.push(document);
                }
                graph.diagnostics.extend(report.diagnostics);
            }
            CandidateKind::AnnotationSource => {
                let report = parse_annotations(&candidate.display_path, &contents)?;
                graph.annotations.extend(report.value);
                graph.diagnostics.extend(report.diagnostics);
            }
        }
    }

    graph.documents.sort_by(|left, right| left.path.cmp(&right.path));
    // After stable path ordering, collapse duplicate document ids by keeping the first
    // document in path order and emitting diagnostics for later duplicates.
    // This makes duplicate handling deterministic regardless of WalkDir traversal order.
    let documents = std::mem::take(&mut graph.documents);
    let mut deduped_documents = Vec::new();
    let mut seen_document_ids = std::collections::BTreeMap::<String, PathBuf>::new();

    for document in documents {
        match seen_document_ids.entry(document.id.clone()) {
            std::collections::btree_map::Entry::Vacant(entry) => {
                entry.insert(document.path.clone());
                deduped_documents.push(document);
            }
            std::collections::btree_map::Entry::Occupied(first_path) => {
                graph.diagnostics.push(Diagnostic {
                    severity: DiagnosticSeverity::Error,
                    path: document.path.clone(),
                    line: None,
                    message: format!(
                        "duplicate markdown id `{}`; first path `{}` wins",
                        document.id,
                        first_path.get().display()
                    ),
                });
            }
        }
    }

    graph.documents = deduped_documents;
    graph
        .annotations
        .sort_by(|left, right| left.path.cmp(&right.path).then(left.line.cmp(&right.line)));
    graph
        .diagnostics
        .sort_by(|left, right| left.path.cmp(&right.path).then(left.line.cmp(&right.line)));

    Ok(graph)
}

#[derive(Debug, Clone, PartialEq, Eq)]
struct ScanCandidate {
    display_path: PathBuf,
    kind: CandidateKind,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
enum CandidateKind {
    Document,
    AnnotationSource,
}

fn is_excluded_dir(root: &Path, entry: &DirEntry) -> bool {
    if !entry.file_type().is_dir() {
        return false;
    }

    if entry.path() == root {
        return false;
    }

    let Some(relative_path) = entry.path().strip_prefix(root).ok() else {
        return false;
    };

    let under_docs = matches!(
        relative_path.components().next(),
        Some(Component::Normal(name)) if name == ".docs"
    );
    let parent_name = entry
        .path()
        .parent()
        .and_then(|path| path.file_name())
        .and_then(|name| name.to_str());
    let file_name = entry.file_name().to_str();

    matches!(
        file_name,
        Some(".git" | "target" | ".idea" | ".vscode" | ".vs" | ".codex" | "node_modules" | "obj")
    ) || matches!(file_name, Some(name) if (name == "tests" || name == "Tests" || name.ends_with(".Tests"))) && !under_docs
        || matches!(file_name, Some("bin")) && parent_name != Some("src")
}

fn classify_path(root: &Path, path: &Path) -> Option<ScanCandidate> {
    let display_path = path.strip_prefix(root).ok()?.to_path_buf();

    match path.extension().and_then(|ext| ext.to_str()) {
        Some("md") if is_indexable_doc(&display_path) => Some(ScanCandidate {
            display_path,
            kind: CandidateKind::Document,
        }),
        Some("rs" | "cs") => Some(ScanCandidate {
            display_path,
            kind: CandidateKind::AnnotationSource,
        }),
        _ => None,
    }
}

fn is_indexable_doc(path: &Path) -> bool {
    matches!(path.components().next(), Some(Component::Normal(name)) if name == ".docs")
}

fn read_failure_diagnostic(path: &Path, kind: ErrorKind, source: std::io::Error) -> Diagnostic {
    let message = match kind {
        ErrorKind::InvalidData => "file is not valid UTF-8".to_string(),
        _ => format!("failed to read file: {source}"),
    };

    Diagnostic {
        severity: DiagnosticSeverity::Error,
        path: path.to_path_buf(),
        line: None,
        message,
    }
}
```

Repository scan contract for this slice:

- skipped directories: `.git`, `target`, `.idea`, `.vscode`, `.vs`, `.codex`, `node_modules`, `obj`, and `tests` directories outside `.docs`, plus generated `bin` trees outside `src/bin`
- skipped test project trees: directories whose final path segment is `tests`, `Tests`, or ends with `.Tests` outside `.docs`, so C# test projects do not leak into the semantic scan
- the `.docs` exception is evaluated from the root-relative path, so `.docs/tests` stays indexable while similarly named trees elsewhere are skipped
- the scan root itself is never excluded even if its basename matches a skipped-directory rule
- explicitly not skipped: `.docs`, `src/bin`, and any other non-excluded directory
- only `.docs/**/*.md` becomes document input
- only `.rs` and `.cs` become annotation input
- all printed paths are stored relative to `--root`
- invalid root and WalkDir entry failures are fatal
- malformed content, invalid UTF-8, and per-file read failures become recoverable diagnostics
- duplicate Markdown `id` values are resolved after stable path ordering so duplicate handling does not depend on WalkDir traversal order
- unsupported file types and out-of-scope Markdown files are ignored silently

## `crates/indexer/src/bin/indexer.rs`

CLI behavior:

- fatal error: exit code `1`
- missing match: exit code `0`
- malformed content and unreadable file reads: still exit code `0`, but print scan diagnostics

```rust
use std::env;
use std::path::PathBuf;

use indexer::{explain, scan_repository, ExplainResult, IndexerError, Result};

fn main() {
    if let Err(error) = run() {
        eprintln!("{error}");
        std::process::exit(1);
    }
}

fn run() -> Result<()> {
    let mut args = env::args().skip(1);

    let Some(command) = args.next() else {
        return Err(IndexerError::cli("usage: indexer explain <id> [--root <path>]"));
    };

    if command != "explain" {
        return Err(IndexerError::cli(format!("unknown command: {command}")));
    }

    let id = args
        .next()
        .ok_or_else(|| IndexerError::cli("missing id for `explain`"))?;

    let mut root = PathBuf::from(".");

    while let Some(arg) = args.next() {
        if arg == "--root" {
            let value = args
                .next()
                .ok_or_else(|| IndexerError::cli("missing value for `--root`"))?;
            root = PathBuf::from(value);
        } else {
            return Err(IndexerError::cli(format!("unexpected argument: {arg}")));
        }
    }

    let graph = scan_repository(&root)?;
    let result = explain(&graph, &id);
    print_result(&result);

    Ok(())
}

fn print_result(result: &ExplainResult) {
    println!("ID: {}", &result.id);
    println!();

    println!("Documents:");
    if result.documents.is_empty() {
        println!("none");
    } else {
        for document in &result.documents {
            match &document.title {
                Some(title) => {
                    println!(
                        "- {} [kind={}, title={}]",
                        document.path.display(),
                        &document.kind,
                        title
                    );
                }
                None => {
                    println!("- {} [kind={}]", document.path.display(), &document.kind);
                }
            }
        }
    }

    println!();
    println!("Annotations:");
    if result.annotations.is_empty() {
        println!("none");
    } else {
        for annotation in &result.annotations {
            match &annotation.role {
                Some(role) => {
                    println!(
                        "- {}:{} [role={}]",
                        annotation.path.display(),
                        annotation.line,
                        role
                    );
                }
                None => {
                    println!("- {}:{}", annotation.path.display(), annotation.line);
                }
            }
        }
    }

    if !result.scan_diagnostics.is_empty() {
        println!();
        println!("Diagnostics:");
        for diagnostic in &result.scan_diagnostics {
            match diagnostic.line {
                Some(line) => {
                    println!(
                        "- {}:{} {}",
                        diagnostic.path.display(),
                        line,
                        &diagnostic.message
                    );
                }
                None => {
                    println!("- {} {}", diagnostic.path.display(), &diagnostic.message);
                }
            }
        }
    }
}
```
