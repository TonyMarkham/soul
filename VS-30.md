# VS: Soul MCP Server

## Context

The Soul indexer core is complete — it scans repos, parses annotations and frontmatter, builds a
semantic graph, and stores it in SQLite. The missing piece is exposing that graph to an LLM. An MCP
server is the bridge: once deployed, Claude Code can query Soul to discover what's documented, what
isn't, and what code has no linked docs — enabling it to write documentation and add annotations
autonomously.

The MCP server is a new `serve` command on the existing `indexer` binary — no new crate, no new
executable. The binary already lives in `.soul/indexer` by convention, so `.mcp.json` points there.
Transport is stdio (JSON-RPC 2.0). Claude Code spawns the process and keeps it alive for the session.

---

## Changes

### 1. `crates/indexer/Cargo.toml`

**No changes needed.** Both the workspace `Cargo.toml` and `crates/indexer/Cargo.toml` already have `rmcp` declared:

- `Cargo.toml` (workspace): `rmcp = { version = "1.4.0", features = ["server", "transport-io"] }`
- `crates/indexer/Cargo.toml`: `rmcp = { workspace = true }`

The `server` feature pulls in `schemars`; `macros` is included via the `default` feature (active since `default-features` is not disabled). No changes to any `Cargo.toml` are required.

> Use `rmcp::schemars` for any `JsonSchema` derives.

## `serve` command (`src/bin/commands/serve.rs`)

Log to **stderr only** — stdout is the JSON-RPC channel.

```rust
use std::path::PathBuf;
use rmcp::{ServiceExt, transport::stdio};
use indexer::{IndexerError, mcp::SoulServer};

pub async fn run(root: PathBuf) -> IndexerResult<()> {
    SoulServer::new(root)
        .serve(stdio())
        .await
        .map_err(IndexerError::mcp)?
        .waiting()
        .await
        .map_err(IndexerError::mcp)?;
    Ok(())
}
```

`IndexerError` has no existing catch-all variant. Add an `Mcp` variant and its `#[track_caller]` constructor to `crates/indexer/src/error/mod.rs`, matching the existing pattern exactly:

```rust
// In the IndexerError enum — add after IndexDb:
#[error("{location} mcp error: {source}")]
Mcp {
    location: ErrorLocation,
    #[source]
    source: Box<dyn std::error::Error + Send + Sync>,
},
```

```rust
// In the impl IndexerError block — add after index_db:
#[track_caller]
pub fn mcp(source: impl std::error::Error + Send + Sync + 'static) -> Self {
    Self::Mcp {
        location: ErrorLocation::from(Location::caller()),
        source: Box::new(source),
    }
}
```

## `SoulServer` (`src/mcp/mod.rs`)

The file begins with `pub mod format;` (the sibling module).

`SoulServer` must implement `Send + Sync` — it does, because `PathBuf: Send + Sync`. rmcp enforces this via the `MaybeSend` bound on `Service`.

```rust
pub mod format;
pub mod explain_params;

use std::path::PathBuf;
use explain_params::ExplainParams;
use rmcp::{
    ServerHandler,
    handler::server::wrapper::Parameters,
    model::{CallToolResult, Content},
    schemars,
    tool, tool_handler, tool_router,
};
use crate::{
    ExplainResult, SemanticGraph,
    IndexerResult,
    config::load_config,
    graph::explain,
    index::{explain_from_index, load_graph, open_index, write_index},
    scan_repository,
};

/// Convert any Display error into a tool-level error result (is_error: true).
/// Not a JSON-RPC protocol error — the call succeeds; the LLM sees the message.
fn err_result(e: impl std::fmt::Display) -> CallToolResult {
    CallToolResult::error(vec![Content::text(e.to_string())])
}

#[derive(Clone)]
pub struct SoulServer { root: PathBuf }

impl SoulServer {
    pub fn new(root: impl Into<PathBuf>) -> Self {
        Self { root: root.into() }
    }
}

#[tool_router]
impl SoulServer {
    #[tool(description = "Scan and index the repository, writing results to .soul/index.db")]
    async fn soul_index(&self) -> CallToolResult {
        self.soul_index_impl().await.unwrap_or_else(err_result)
    }

    #[tool(description = "Return full documentation and annotations for a Soul ID")]
    async fn soul_explain(
        &self,
        Parameters(p): Parameters<ExplainParams>,
    ) -> CallToolResult {
        self.soul_explain_impl(p).await.unwrap_or_else(err_result)
    }

    #[tool(description = "List all indexed documents with their ID, kind, title, and path")]
    async fn soul_list_documents(&self) -> CallToolResult {
        self.soul_list_documents_impl().await.unwrap_or_else(err_result)
    }

    #[tool(description = "List annotation IDs with no matching document, and document IDs with no annotation")]
    async fn soul_list_gaps(&self) -> CallToolResult {
        self.soul_list_gaps_impl().await.unwrap_or_else(err_result)
    }
}

#[tool_handler(name = "soul")]
impl ServerHandler for SoulServer {}

// ---- private impl (use ? freely) ----

impl SoulServer {
    async fn soul_index_impl(&self) -> IndexerResult<CallToolResult> {
        let config = load_config(&self.root)?;
        let graph = scan_repository(&self.root, &config)?;
        write_index(&self.root, &graph).await?;
        Ok(CallToolResult::success(vec![Content::text(format!(
            "Indexed {} documents, {} annotations, {} diagnostics.",
            graph.documents.len(), graph.annotations.len(), graph.diagnostics.len()
        ))]))
    }

    async fn soul_explain_impl(&self, p: ExplainParams) -> IndexerResult<CallToolResult> {
        let result = match open_index(&self.root).await? {
            Some(pool) => explain_from_index(&pool, &p.id).await?,
            None => {
                let config = load_config(&self.root)?;
                let graph = scan_repository(&self.root, &config)?;
                explain(&graph, &p.id)
            }
        };
        Ok(CallToolResult::success(vec![Content::text(format::explain_result(&result))]))
    }

    async fn soul_list_documents_impl(&self) -> IndexerResult<CallToolResult> {
        let graph = self.load_graph_or_scan().await?;
        let lines: Vec<String> = graph.documents.iter()
            .map(|d| format!("{} [{}] {} — {}",
                d.id, d.kind,
                d.title.as_deref().unwrap_or("(untitled)"),
                d.path.display()))
            .collect();
        Ok(CallToolResult::success(vec![Content::text(lines.join("\n"))]))
    }

    async fn soul_list_gaps_impl(&self) -> IndexerResult<CallToolResult> {
        let graph = self.load_graph_or_scan().await?;
        Ok(CallToolResult::success(vec![Content::text(format::gaps(&graph))]))
    }

    async fn load_graph_or_scan(&self) -> IndexerResult<SemanticGraph> {
        match open_index(&self.root).await? {
            Some(pool) => load_graph(&pool).await,
            None => {
                let config = load_config(&self.root)?;
                scan_repository(&self.root, &config)
            }
        }
    }
}
```

## `format.rs` (`src/mcp/format.rs`)

Plain-text formatters (LLM-readable). Full implementations:

```rust
use crate::model::{CodeAnnotation, Document};
use crate::graph::ExplainResult;
use crate::model::SemanticGraph;
use std::collections::BTreeSet;

pub fn explain_result(result: &ExplainResult) -> String {
    let mut out = String::new();
    out.push_str(&format!("# Soul ID: {}\n\n", result.id));
    if result.documents.is_empty() && result.annotations.is_empty() {
        out.push_str("No documents or annotations found for this ID.\n");
        return out;
    }
    if !result.documents.is_empty() {
        out.push_str("## Documents\n\n");
        for doc in &result.documents {
            out.push_str(&document(doc));
            out.push('\n');
        }
    }
    if !result.annotations.is_empty() {
        out.push_str("## Annotations\n\n");
        for ann in &result.annotations {
            out.push_str(&annotation(ann));
            out.push('\n');
        }
    }
    out
}

pub fn document(doc: &Document) -> String {
    format!(
        "- [{}] {} — {}\n  Title: {}",
        doc.kind,
        doc.id,
        doc.path.display(),
        doc.title.as_deref().unwrap_or("(untitled)"),
    )
}

pub fn annotation(ann: &CodeAnnotation) -> String {
    format!(
        "- {} (role: {}) @ {}:{}",
        ann.id,
        ann.role.as_deref().unwrap_or("(none)"),
        ann.path.display(),
        ann.line,
    )
}

pub fn gaps(graph: &SemanticGraph) -> String {
    let doc_ids: BTreeSet<&str> = graph.documents.iter().map(|d| d.id.as_str()).collect();
    let ann_ids: BTreeSet<&str> = graph.annotations.iter().map(|a| a.id.as_str()).collect();

    // Annotation IDs that have no matching document
    let unlinked: Vec<&str> = ann_ids.iter().copied().filter(|id| !doc_ids.contains(id)).collect();
    // Document IDs that have no matching annotation
    let undocumented: Vec<&str> = doc_ids.iter().copied().filter(|id| !ann_ids.contains(id)).collect();

    let mut out = String::new();
    out.push_str(&format!("## Unlinked Annotations ({})\n", unlinked.len()));
    for id in &unlinked {
        out.push_str(&format!("  {id}\n"));
    }
    out.push_str(&format!("\n## Undocumented IDs ({})\n", undocumented.len()));
    for id in &undocumented {
        out.push_str(&format!("  {id}\n"));
    }
    out
}
```
